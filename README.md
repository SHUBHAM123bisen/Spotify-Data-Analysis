# Spotify-Data-Analysis

- ## Overview
This project analyzes Spotify listening history to understand music consumption patterns, including albums, artists, tracks, and listening behaviour across days, hours, and years.
It includes an interactive Power BI dashboard with drill-downs, filters, and a detailed grid view.
